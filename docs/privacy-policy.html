Privacy Policy

Last Updated: November 26, 2025

1. Overview

This Internal Reporting System ("the Application") is designed for internal business use only. The Application integrates with QuickBooks, Mercury Banking, and GitHub to provide consolidated reporting and analytics.

2. Data Collection

The Application accesses and processes the following data:

  • QuickBooks: Invoice data, payment records, and accounts receivable information
  • Mercury Banking: Account balances and transaction history
  • GitHub: Commit history and pull request data

3. Data Storage

All data is stored securely in Supabase (PostgreSQL database) with the following protections:

  • Encrypted connections (SSL/TLS)
  • Access restricted to authorized personnel only
  • Regular automated backups

4. Data Usage

Data collected by the Application is used solely for:

  • Internal business reporting and analytics
  • Weekly automated reports delivered via Slack
  • Financial and development metrics tracking

5. Data Sharing

We do not share, sell, or distribute your data to any third parties.

Data is only accessible to:

  • Authorized employees of the company
  • The Application's secure database infrastructure (Supabase)

6. Data Retention

Data is retained indefinitely for historical reporting purposes. You may request data deletion at any time by contacting the system administrator.

7. Security

We implement industry-standard security measures including:

  • OAuth 2.0 authentication for QuickBooks access
  • API key authentication for Mercury and GitHub
  • Environment variable protection for sensitive credentials
  • Encrypted data transmission

8. Your Rights

As a user of this Application, you have the right to:

  • Access your data at any time
  • Request data corrections or updates
  • Request data deletion
  • Revoke Application access to QuickBooks

9. Changes to This Policy

We may update this Privacy Policy from time to time. The "Last Updated" date at the top of this page indicates when the policy was last revised.

10. Contact

For questions about this Privacy Policy or data handling practices, please contact your system administrator.
